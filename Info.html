ESP32

ESP32 is a microcontroller used as the main controller of our project. It has built-in WiFi and Bluetooth which helps in IoT applications.

MQ Smoke Sensor

The MQ sensor is used to detect smoke and harmful gases. It sends data to the ESP32 for processing and alerts.

MPU6050 Sensor

MPU6050 is a motion sensor that measures acceleration and rotation. It is used to detect movement and orientation.

Power Supply

The power supply provides required voltage to all components. It ensures stable operation of the system.
